# PickMyAI Roadmap

## P0 — MVP Category-Based Tournament (Current)
Dynamic category-based tournament with two modes (Fast Track / Deep Dive), 2-5 selectable categories, voting, tiebreaker, and per-category assessment framework.
See: [tournament-comparison.md](tournament-comparison.md)

## P1 — Image Generation Category
Enable the Image Generation category (currently "coming soon"). Only AIs with image capability participate (ChatGPT, Gemini, Grok, Meta AI). Claude and Mistral shown greyed out with "Does not offer image generation" indicator.

## P2 — Video Generation Category
Enable the Video Generation category. Fewer AIs support this — greyed-out pattern for non-capable platforms.

## P3 — Music Generation Category
Enable the Music Generation category. Same greyed-out pattern.

## P4 — Conversation Deep Linking
Research and implement "Try it free" that opens the selected AI platform with the conversation pre-loaded (prompt + response). Currently no major platform supports this via URL.
